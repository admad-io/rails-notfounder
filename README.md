# RailsNotFounder
Aggregate 404s in a separate model and email weekly reports with 404s per domain
request.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'rails_not_founder'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install rails_not_founder
```
